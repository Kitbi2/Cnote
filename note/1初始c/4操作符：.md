## 逻辑操作符：

1.&&：经常用于while或者if，俩个条件为真是判断条件才成立

```
int main()
{
	if(a<0&&b>0)
	{
		printf("hh");
	}
}
```

上面那串代码只有在a<0和b>0同时成立的时候才会打印hh

```c
int main()
{
	if(a<0||b>0)
	{
		printf("hh");
	}
}
```

上面那串代码只有在a<0和b>0同时成立的时候才会打印hh；

```c
int main()
{
	if(a<0&&b>0)
	{
		printf("hh");
	}
}
```

上面那串代码只要在a<0和b>0成立一个就会打印hh；

这里还有一个点，就是短路问题，之后再说吧。

## 条件操作符：exp1?exp2:exp3

这个语句意识是当exp1成立的时候就会执行exp2，当exp1不成立就执行exp3
